Documentation

This project visualizes food and drink establishments in NYC opened in 2025, using a CSV data source and Leaflet.js for mapping.

How to Use

  • Pan and zoom the map to explore locations.
  • Click on emoji pins to view details and images in popups.
  • Use the filters and search to narrow down results.

Data Fields

  • Name: The establishment's name
  • Category: Pulled from tags.name
  • Website: Link to the official site
  • Image: Photo of the establishment
  • Address: Street address and postcode
  • Verified on: Date the place was verified open

About the Data

Data is sourced from livexyz.com and is current as of 2025 (so far).
