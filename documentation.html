Documentation

This project visualizes food and drink establishments in NYC opened in 2025, using a CSV data source and Leaflet.js for mapping. It's a project for the Software Design Lab class at CUNY's Graduate Center.

About the Data

Data is sourced from livexyz.com and the dates displayed indicate when the scouting team discovered and documented the business, so are likely a few months removed from the establishment's actual opening date. Data displayed were verified between 1/1/25 and 5/4/25, and filtered for the categories "food" and "drinks". There are 904 points.

Data Fields

  • Name: The establishment's name
  • Category: Subcategory of food or drinks establishment
  • Website: Link to the business's website, if available
  • Image: Photo of the establishment
  • Address: Street address and postcode
  • Verified on: Date the place was verified open
